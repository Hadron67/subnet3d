# Subnet3d - 3d版的Subnet
用blender做的Subnet的三维模型。准备做成3D版的Subnet。

3D models and scenes in Subnet made with blender.I'm planning making a 3D version of the Subnet.

## 目录结构
./common下面放的是很多场景里面重复出现的模型，如传送机等，./locations里面是各个场景的目录。
